This directory contains some basic examples and timings compared
to existing packages such as `iisignature`, `signatory` and `esig`.

- `signatureExample.py` - a simple example of signatures
- `sigKernelExample.py` - a simple example of signature kernels
- `signatureTimesPlot.py` - plots elapsed time vs degree for signatures
- `sigKernelTimesLen.py` - plots elapsed time vs path length for signature kernels
- `sigKernelTimesDim.py` - plots elapsed time vs path dimension for signature kernels
