This directory contains some basic examples and timings compared
to existing packages such as `iisignature`, `signatory`, `esig`
and `sigkernel`.

- `signature_example.py` - a simple example of signatures
- `sig_combine_example.py` - a simple example of combining signatures
- `sig_kernel_example.py` - a simple example of signature kernels


- `sig_times.py` - runtime for signatures
- `sig_backprop_times.py` - runtime for backprop through signatures
- `sig_kernel_times.py` - runtime for signature kernels
- `sig_kernel_backprop_times.py` - runtime for backprop through signature kernels


- `sig_times_plot.py` - plots runtime vs degree for signatures
- `sig_backprop_times_plot.py` - plots runtime vs degree for backprop through signatures
- `sig_kernel_times_plot.py` - plots runtime vs path length for signature kernels
- `sig_kernel_backprop_times_plot.py` - plots runtime vs path length for backprop through signature kernels
