<h1 align='center'>cpSIG and cuSIG</h1>
<h2 align='center'>C++ Libraries for Signature Computations on CPU and GPU</h2>

This directory contains source and project files for the cpSIG and cuSIG C++ libraries.
The directories `cpsig_unit_testing` and `cusig_unit_testing` provide a limited number of
unit tests for the libraries, with more advanced testing provided via the python wrapper
`pysiglib`.
